# Deep Learning – Goodfellow, Bengio & Courville

This repository is a structured study and implementation companion for the book  
**Deep Learning** by Ian Goodfellow, Yoshua Bengio, and Aaron Courville.

## Objectives
- Build strong theoretical foundations in deep learning
- Re-derive core algorithms from scratch
- Implement key models using PyTorch
- Run controlled experiments to understand optimization and generalization

## Repository Structure
- `notes/` – Chapter summaries and explanations
- `math/` – Mathematical prerequisites and derivations
- `code/` – Implementations (from scratch + PyTorch)
- `experiments/` – Empirical studies and ablation experiments
